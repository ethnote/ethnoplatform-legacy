# Notes

The fieldnotes have different view it can be displayed in:

- [**List:**](#list) A list of all the fieldnotes in the project
- [**Attachments:**](#attachments) A list of all the attachments in the project
- [**Hashtags:**](#hashtags) A list of all the hashtags in the project
- [**Mentions:**](#mentions) A list of all the mentions in the project
- [**Map:**](#map) A map of all the fieldnotes in the project

Accross the different views, a search word can be entered to filter the fieldnotes. The search word is matched against the title, athour, and text of the fieldnotes. A filter button reveals a filter menu where the fieldnotes can be filtered by:

- Author
- Template variant

## List

### Sort List

The list view shows all the fieldnotes in the project. The fieldnotes can be sorted by:

- Date updated
- Date created
- Author (full name or email)
- Title

### Download Notes

The individual fieldnotes can be selected by clicking on the checkbox on the left side of the fieldnote. It's possible to select all of them by clicking on the checkbox in the top left corner on top of the list. The selected fieldnotes can be deleted or downloaded by clicking on the delete-icon or "Download Selected".

The fieldnotes can be downloaded in different formats. The formats available are:

- .docx (Word)
- .xlsx (Excel)
- .csv
- .JSON
- .txt (Raw text)

The fieldnotes can either be downloaded in one single file, or in a folder structure, where each folder represents a fieldnote. The folder structure will also include the attachments for that fieldnote.

### Create New Note

A new fieldnote can be created by clicking on "New Note" in the top right corner. A dialog is shown where the title and template variant can be selected. A title and template variant is required to create a new fieldnote.

## Attachments

The attachments view shows all the attachments in the project.

### Sort Attachments

The attachments can be sorted by:

- Date created
- File type
- Note title
- Author (full name or email)

### Filter Attachments

The files can also be filtered by file type. The files are already filtered by the search word and filter button, that are common for all the views.

### Download Attachments

The visible files can be downloaded as a ZIP-file.

## Hashtags

All hashtags that are used in the project will be listed in the hashtags view. The top sections showns an overview of all the hashtags that are used together with the notes they are used in. The bottom sections shows each hashtag in an accordian view, where the hashtag is presented in the context it's used in. If more context is needed, the arrow buttons can be used to include more or fewer sentences before and after the hashtag.

### Download Hashtags

Each of the hashtags in the bottom section can be selected (or all at once) and downloaded in different formats. The formats available are:

- .docx (Word)
- .xlsx (Excel)
- .csv
- .JSON
- .txt (Raw text)

Hashtags are used for topics, and are written as #topic.

## Mentions

Mentions works exactly like hashtags, but instead of showing the hashtags used in the project, it shows the mentions used in the project. Mentions are used for people, and are written as @name.

## Map

If any location data is available in the note, it will be shown on the map. Each location is represented by a marker on the map. The markers can be clicked to enter the fieldnote.
